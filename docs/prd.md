# Apartment Cleaning Routine App PRD

Version: 0.2, founder answers integrated

This repository implements the MVP described in the uploaded PRD:

- Public mobile-first website.
- No login required.
- Cleaning routines only.
- Public starter templates.
- Today screen with morning, afternoon, and evening blocks.
- Each routine block has an independent 0% to 100% completion score.
- Task progress and settings persist in localStorage on the current device.
- A fresh checklist is created for each local cleaning day.
- Apartment zones are visible and manually selectable.
- A 15-minute zone timer supports start, pause, resume, and reset.
- Local data can be cleared from Settings.

Version 0.2 Google backup is intentionally not included in this MVP.
